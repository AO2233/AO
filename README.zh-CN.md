## 特性

- 支持移动设备样式  
- 深色/浅色模式自动切换（基于系统主题模式切换）：
  - Windows 10+ ✅  
  - macOS ✅  
  - Linux Gnome 43 ✅  
  - Android ✅  
  - Meta Quest ✅  
- 完整 LaTeX 支持  
- 嵌入式 PDF 显示（不支持 iOS WebKit 浏览器）  
- 音乐播放器（支持 APlayer、Spotify）  
- 视频播放器（支持 DPlayer、YouTube、Bilibili）  
- 可折叠内容框
